A simple climbing session tracker. Log attempts, tops, session flashes, and flashes across different grades, then track your progress with charts and session history. The app uses a weighted scoring system where attempts = 0.25×grade, tops = 1×grade, session flashes = 1.5×grade, and flashes = 2×grade, making harder climbs and cleaner ascents worth more points. All data is stored locally in your browser.
